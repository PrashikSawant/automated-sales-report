# Automated Sales Report

A Python script that reads raw Superstore sales data and automatically 
generates a formatted Excel report with 5 sheets.

## KPIs Covered
- Total Revenue
- Revenue by Region
- Top 5 Products
- Monthly Revenue Trend
- Revenue by Category

## Tools Used
- Python
- Pandas
- Openpyxl
- Jupyter Notebook

## Output
Formatted Excel file with color-coded headers and multiple sheets.
